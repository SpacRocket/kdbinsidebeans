.Q.id x

Where x is

  • a symbol atom, returns x with items sanitized to valid q names

    
q).Q.id each `$("ab";"a/b";"two words";"2drifters";"2+2")
`ab`ab`twowords`a2drifters`a22

q).Q.id flip (5#.Q.res)!(5#())
